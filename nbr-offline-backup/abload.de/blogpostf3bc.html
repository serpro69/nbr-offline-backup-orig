Tool

geschrieben von Markus, 29.09.2008 09:49 Uhr

Es wird im Moment unter anderem durch einen neuen Toolprogrammierer an einer neuen, auf Java basierenden Version des Abloadtools gearbeitet.

Diese wird hier und da anders - und intuitiver und damit bequemer - funktionieren als die aktuelle Beta-Version.

Die Entwicklung dessen ging zugegebenermaßen leider nur sehr träge. Parallel dazu versucht der Programmierer des aktuellen Abloadtools eine Verbesserung seiner Variante, die wohl als Alternative ebenfalls zum Download angeboten werden wird.

So kann der User frei entscheiden, auf welches Programm er setzen möchte.

Es wird ein Newsletter verschickt, sobald mindestens eine der beiden Versionen online gestellt wird.
