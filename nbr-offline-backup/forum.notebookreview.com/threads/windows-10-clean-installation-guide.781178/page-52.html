    1) DWS is optional and not a must. I use it to clean up SXS folders, uninstall all updates, permanently delete the Windows Store apps leftovers of the ones I've uninstalled. No need to run it on your machine, this does not yield any speed benefits and can cause problems if you ever wanted to uninstall a particular buggy update OR wanted to refresh your Windows without formatting it.

    2) Not if you use O&O ShutUp10. Here is the last version of the Desktop Skype as now only the Windows store crippled version is offered: Skype v7.40

    3) NOD32 is still the recommended Antivirus. It has a built in HTTP Scanner to intercept any bad sites or malicious file downloads. The Windows Firewall is all you need.

    BUT, if you don't have a license for NOD32 yet, then buy ESET Internet Security as that includes a firewall which has more control than the Windows Firewall and doesn't have an allow rule by default to all programs like the Windows Firewall does.
