    How to stop Windows 10 from installing unwanted Store apps

    Windows 10 automatically install unwanted apps from the Windows store without your consent such as games or other apps (ie. Candy Saga Crash, Mickey Mouse, etc.).

    To stop this, do the following:

    1) Press the Windows Key + i to launch PC Settings.

    2) Go to Perosnalization > Start then uncheck "Show suggestions occasionally in Start"

    3) Download the following ZIP fill which contains to registry files, one to disable and the other is to re-enable the automatic installation of Windows Store Apps: Download Windows Store App Automatic Installation

    4) Double click on the file named Disable automatically installing suggested apps.reg to merge it with the registry

    5) Uninstall any apps that you don't want which were automatically installed from the Windows Store

    6) Reboot
